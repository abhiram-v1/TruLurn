# TruLurn Presentation Showcase

This folder preserves the presentation-friendly TruLurn frontend separately from the main product build.

Use this when you want to show the idea visually to someone without opening a slide deck.

## Run It

```bash
npm install
npm run dev -- --port 3001
```

Then open:

```text
http://localhost:3001
```

## Notes

- This is a saved showcase copy, not the main product workspace.
- It uses mock data and local UI only.
- Keep the main project focused on the actual usable TruLurn product experience.
